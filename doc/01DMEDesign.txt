DME-Core

The DME-Core is a RISC OS App that contains both logic and resources to achieve the requirement goals.

!DMECore
├── Resources
│   └── UK
└── Store
    ├── Controls
    ├── Gadgets
    └── Themes

The Resources directory is the standard RISC OS repository for territory and messages as well as app templates (if neeved)

The Store directory contains all the Global Controls, Gadgets and Themes repositories.

The DMECore !RunImage will read the configurations stored in Boot:Choices.DME and apply all the changes (in the futures it may also apply RISC OS Patches required by advnaced functionalities)

The DMECore !RunImage will detect the RISC OS release being used by the user and will apply setting saccordingly.
